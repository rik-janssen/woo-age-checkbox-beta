== Description ==
This plugin adds an age check to the checkout form. Customers have to tick a box in order to be able to place an order. 
You can customise the messages and enable/disable the function from the Woocommerce settings page.
Works like a charm with Wordpresss 5.3 and the latest versions of Woocommerce.

== Installation ==
1. Upload the unpacked folder to the "/wp-content/plugins/" directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to the settings menu and activate the plugin.

== Frequently Asked Questions ==
= Where can I find the settings? =
In the wp-admin bar go to Woocommerce > settings and you will find the settings just below coupon settings, in the general settings tab.
